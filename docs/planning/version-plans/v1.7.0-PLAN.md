# Version 1.7.0 Release Plan - Platform & API

## Overview

**Release Codename:** "Developer Platform"
**Target Release Date:** Q4 2026 (December 2026)
**Theme:** API-First Platform & Developer Experience
**Dependencies:** v1.5.0 (User accounts, authentication)

This release transforms InfoGraphix AI into a developer platform by introducing comprehensive REST APIs, webhook systems, official SDKs, and developer tools. The goal is to enable third-party integrations and programmatic infographic generation.

---

## Features

### Feature 1: REST API v1

**Priority:** High
**Effort:** Very High
**Dependencies:** v1.5.0 (Authentication system)

**Description:**
Comprehensive REST API for programmatic infographic generation, management, and retrieval. Includes authentication, rate limiting, quota management, and comprehensive documentation.

**Acceptance Criteria:**
- [ ] Complete REST API for all infographic operations
- [ ] OpenAPI 3.0 specification
- [ ] Authentication via API keys
- [ ] Rate limiting (100 requests/hour free, higher for paid)
- [ ] Quota management (generations/month)
- [ ] Comprehensive error responses
- [ ] Versioned endpoints (/api/v1/)
- [ ] CORS configuration for web apps
- [ ] API playground for testing

**API Endpoints:**
```
POST   /api/v1/generate              # Generate infographic
GET    /api/v1/generations/:id       # Get generation status/result
GET    /api/v1/generations           # List all generations
DELETE /api/v1/generations/:id       # Delete generation
GET    /api/v1/templates              # List templates
POST   /api/v1/batch                  # Create batch generation
GET    /api/v1/batch/:id              # Get batch status
GET    /api/v1/account/usage          # Get usage statistics
POST   /api/v1/webhooks               # Create webhook
GET    /api/v1/webhooks               # List webhooks
DELETE /api/v1/webhooks/:id          # Delete webhook
```

- [ ] Implement all API endpoints

**Package:** `@infographix/sdk` on npm

**Acceptance Criteria:**
- [ ] Full TypeScript support with type definitions

- [ ] Webhook signature verification helpers
- [ ] Comprehensive JSDoc documentation
- [ ] 20+ code examples
- [ ] Unit test coverage >90%
- [ ] Bundle size <50KB (gzipped)

**SDK Features:**
```typescript
import { InfoGraphixClient } from '@infographix/sdk';

const client = new InfoGraphixClient({ apiKey: 'xxx' });

// Generate infographic
const result = await client.generate({
  topic: 'AI in Healthcare',
  style: 'modern',
  palette: 'professional',
  size: '2K'
});

// Batch generation
const batch = await client.batch.create([
  { topic: 'Topic 1' },
  { topic: 'Topic 2' }
]);

// Templates
const templates = await client.templates.list();
```
